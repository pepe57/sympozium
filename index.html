sympozium

A coordination layer for multi-agent AI systems on Kubernetes

Agents are Pods, policy is CRDs — and when an agent needs a model, it claims one; Sympozium never decides where it runs. Placement belongs to llmfit-dra; serving belongs to the engine. Why the boundary matters.

Install

curl -fsSL https://deploy.sympozium.ai/install.sh | sh

Deploy to your cluster

sympozium install
